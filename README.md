# Vertex is a small, custom programming language created as a hobby project. This repository contains the interpreter for Vertex, written in C++.

## Features: 
### 1. Executes Vertex source files (files with extension .vx)
### 2. Supports basic arithmetic, integer and string variable support, and conditionals
### 3. Minimal and fast -> Fast for learning language design
### 4. Descriptive error messages for logical and syntactical errors

# Usage
Vertex supports REPL mode, and can also read and execute from files. File extensions must be ".vx".
##### Syntax for using REPL mode is given below.  

> ./vertex
 
##### Syntax for using File mode is given below.

> ./vertex [FILENAME]
